## 🤝 介绍

- 本插件利用Onebot协议接口，实现了QQ群的各种管理功能，旨在帮助用户更方便地管理群聊。  
- 功能包括：禁言、全体禁言、踢人、拉黑、改群昵称、改头衔、设管理员、设精华、撤回消息、改群头像、改群名、发群公告等等。
- 插件提供前端配置面板，支持默认群模板、按群独立配置、群头像与群名展示等能力。
- 权限模型：仅超级管理员（在插件配置 `super_admins` 中指定的QQ号）可使用所有群管指令，非超级管理员的指令消息会被静默忽略。Bot 仍需具备群主/管理员角色才能执行相应操作。

## 📦 安装

在astrbot的插件市场搜索astrbot_plugin_qqadmin，点击安装即可  

## ⌨️ 使用说明

### 权限说明

- **超级管理员**：在插件配置项 `super_admins` 中填入QQ号，多个QQ号用换行或逗号分隔。只有超级管理员才能使用群管插件的所有指令。
- 超级管理员配置与 AstrBot 自身的管理员配置（`admins_id`）完全独立，互不影响。
- 非超级管理员发送的指令消息会被静默忽略（不回复、不执行）。
- Bot 本身仍需在群内具备相应角色（群主/管理员）才能执行禁言、踢人等操作。
- 超级管理员免受违禁词检测和刷屏检测的自动处罚。

### 指令格式

- 所有指令直接发送消息文本即可触发，无需添加任何前缀。
- 指令名称即为消息的第一个词，后续内容作为参数。

## 🤝 配置

进入 AstrBot 插件设置页后，可直接使用本插件自带的前端配置面板进行管理。

### 必须配置项

| 配置项 | 类型 | 说明 |
|--------|------|------|
| `super_admins` | list | 超级管理员QQ号列表。只有此处配置的QQ号才能使用群管插件的所有指令。多个QQ号用换行或逗号分隔。与AstrBot的管理员配置完全独立。**必须配置，否则所有命令无法执行。** |

### 全局配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `admin_audit` | bool | false | 开启后进群事件仅通知bot管理员，不再发送到群聊 |
| `random_ban_time` | string | "30~300" | 随机禁言时间范围（秒），用~分隔 |
| `vote_ban.ttl` | int | 120 | 投票禁言持续时长（秒） |
| `vote_ban.threshold` | int | 3 | 投票通过所需票数 |
| `llm_get_msg_count` | int | 10 | 取名指令获取消息的默认轮数（每轮200条） |
| `level_threshold` | int | 50 | 高等级成员阈值 |

### 默认群配置项（按群可覆盖）

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `join_switch` | bool | true | 进群审核总开关 |
| `join_min_level` | int | 8 | 进群等级门槛（0=不限制） |
| `join_max_time` | int | 3 | 进群最大尝试次数 |
| `join_accept_words` | list | [] | 进群白词 |
| `join_reject_words` | list | [] | 进群黑词 |
| `join_no_match_reject` | bool | false | 未命中白词自动驳回 |
| `reject_word_block` | bool | false | 命中黑词自动拉黑 |
| `block_ids` | list | [] | 进群黑名单 |
| `join_welcome` | string | "" | 进群欢迎词（支持 {nickname}） |
| `join_ban_time` | int | 0 | 进群禁言时长（秒，0=不禁言） |
| `leave_notify` | bool | false | 退群通知 |
| `leave_block` | bool | false | 退群自动拉黑 |
| `builtin_ban` | bool | false | 内置违禁词检测 |
| `custom_ban_words` | list | [] | 自定义违禁词 |
| `word_ban_time` | int | 360 | 违禁词禁言时长（秒，0=不禁言） |
| `spamming_ban_time` | int | 0 | 刷屏禁言时长（秒，0=不启用） |

### 前端面板说明

在Astrbot插件页面，点击插件logo图进入详情页面，在插件行为下点击"打开"，即可进入群管插件的前端配置面板。

- 左侧为群列表，支持按群名或群号搜索，也可以手动同步最新群列表。
- 右侧为当前选中群的配置表单，顶部显示当前群名称，下方直接展示可编辑配置项。
- "默认群"就是全局默认模板。新群或未单独配置的群，会优先继承默认群配置。
- 每个具体群的第一项配置为"跟随默认配置"。开启时继承默认群，关闭时保存独立配置。
- 切换群时，当前页面的修改会自动触发保存。
- 面板会显示群头像、群名称和群号；群名称会优先通过 QQ 接口获取并缓存。
- 如果某个群在实时列表中显示为 0 人，插件会视为该群已失效并自动清理。
- 面板右上角提供独立的浅色/深色/自动主题切换按钮。

### 配置思路

- 先把通用规则配置在"默认群"中，作为全局模板。
- 仅对少数特殊群关闭"跟随默认配置"，再按需覆盖个别设置。
- 大多数群如果保持跟随默认配置，后期维护会轻松很多。
